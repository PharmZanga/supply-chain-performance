# 2025 Supply Chain Performance Dashboard

Professional React dashboard for Zambia public health logistics performance monitoring.

## Run locally

```bash
npm install
npm run dev
```

## Build for GitHub Pages

```bash
npm run build
```

The Vite base path is configured for:

https://pharmzanga.github.io/supply-chain-performance/

## Data replacement

Sample 2025 data lives in `src/mockSupplyChainData.js`. Replace the generated records in that file with parsed CSV/Excel data while keeping the same field names, or map your real source columns to the existing schema before rendering.
